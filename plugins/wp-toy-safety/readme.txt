Toy Safety Regulation (revising Dir. 2009/48/EC) workbench: toy register, restricted substances, conformity assessment, Digital Product Passport, RAPEX/Safety Gate.

== Description ==

Free tier: toy register with GTIN + age range + ≤ 36 months flag + CE mark + EU Declaration of Conformity URL; restricted-substance register (CMR / endocrine / PFAS / phthalates / heavy metals / nitrosamines / 55 fragrances / formaldehyde / aromatic amines / bisphenols); conformity-assessment register across modules A / Aa / B+C / B+E with notified-body NB number + cert number + validity; Safety Gate (RAPEX) incident register with operator-configurable initial + follow-up windows; economic-operator chain (manufacturer / authorised rep / importer / distributor / fulfilment / online marketplace); per-toy DPP builder (XML `urn:toy:eurocomply:0.1` + JSON); 10-tab admin; CSV export.

= Pro features =

* Live Safety Gate / RAPEX submission with statutory deadline timer
* Signed PDF EU Declaration of Conformity (XAdES / PAdES)
* DPP backbone registry sync (ESPR Digital Product Passport)
* GTIN / EAN barcode + DataMatrix / QR generator on each DPP
* Notified-body NANDO directory cache with automatic expiry alerts
* Annex II Appx. C 55-fragrance allergen scanner against ingredient lists
* WooCommerce product meta — auto-stamp DPP on shop pages
* EU 2019/1020 fulfilment-service-provider duty checks
* Bulk CSV import of toy register + per-row substance limits
* REST + webhooks
* Recall-management workflow
* WPML / Polylang for multi-language warnings (Art. 11)
* Multi-site network aggregator
* 5,000-row CSV cap
