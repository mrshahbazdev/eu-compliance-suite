# Translations

The `.pot` file for this plugin will be generated by WP-CLI at build time:

```bash
wp i18n make-pot . languages/eurocomply-legal.pot
```

WordPress.org will then auto-generate per-locale `.po` / `.mo` files via
[translate.wordpress.org](https://translate.wordpress.org/) once the plugin
is approved.

Target locales for v1.0:
- `de_DE`, `de_AT`, `de_CH` (source)
- `en_US` (second source)
- `fr_FR`, `nl_NL`, `it_IT`, `es_ES`
