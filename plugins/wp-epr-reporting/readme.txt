Extended Producer Responsibility (EPR) data capture + multi-country reporting for WooCommerce stores selling into the EU.

== Description ==

Capture per-country EPR registration numbers (France Triman / Agec, Germany LUCID,
Spain, Italy CONAI, Netherlands Afvalfonds, Austria ARA, Belgium Fost Plus) and
packaging material weights on every WooCommerce product, then produce the
audit-ready CSVs that each national registry expects.

Free tier:

* Per-country registration numbers on the product metabox.
* Packaging weight declaration (paper, plastic, glass, metal, wood, composite, other).
* Shop-wide default registrations with per-product override.
* Admin dashboard showing compliant / warning / missing per-country status.
* CSV export per country (and a combined CSV).

Pro tier (stub — coming soon):

* Automated registry submission (LUCID, CITEO, CONAI, Afvalfonds).
* Per-registry code mapping.
* Batch/lot recall workflow.
* Multi-producer mode for stores representing multiple brands.
* 10-year audit archive.

Part of the EuroComply EU compliance plugin suite.
