Reg. (EU) 2023/1114 (MiCA): asset register (ART/EMT/other), white-paper notification + 12-day standstill (Art. 8), Art. 7 marketing log, Art. 31 complaint handling, Art. 87/88 disclosures.

== Description ==

Free tier provides Articles 6 / 7 / 8 / 17 / 19 / 27 / 31 / 51 / 87 / 88 admin-side workflows. ART/EMT regime applies from 30 June 2024; rest of MiCA from 30 December 2024.

= Pro features =

* Live NCA submission of white papers + standstill timer
* Reserve-of-assets snapshot scheduling for ART/EMT issuers (Art. 36–38)
* Authorisation register sync (CASP / ART / EMT) — auto-pull ESMA register
* Signed PDF white papers + audit bundle
* REST + webhooks
* Marketing-comms compliance scanner
* Complaint-handling SLAs with Slack / Teams alerts
* Insider-information delay-justification workflow (Art. 88)
* Suspicious-order / market-abuse rule engine
* EBA significant-token threshold tracker (Art. 43)
* WPML / Polylang
* Multi-site network aggregator
* 5,000-row CSV cap
