# EuroComply — EU Compliance Plugin Suite

EU-specific compliance plugins for WordPress, WooCommerce, Shopify, Chrome, Figma, and VS Code.
Built to help EU-27 merchants and service providers meet regulations like GDPR, GPSR, EAA, EU e-invoicing mandates, EPR, DSA, and more — without paying €500+/month to specialists.

> **Status:** Phase 1 in progress. Plugin #1 (Legal Pages / Impressum) is the first shipping component.

## Repository layout

```
eu-compliance-suite/
├── plugins/
│   ├── wp-legal-pages/        # Plugin #1 — Impressum / AGB / Datenschutz / Widerruf (WordPress)

│   └── wp-eu-vat/             # Plugin #3 — EU VAT + OSS (Phase 1)
```
